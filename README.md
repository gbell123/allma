# allma exercise

## Prerequisites:
```
docker
yarn
```

## Setup and Run the Example:
```
git clone git@github.com:gbell123/allma.git;
cd allma;
yarn install;
docker-compose up;
```

## "Vanilla" if you have any issues:
```
git clone git@github.com:gbell123/allma.git;
cd allma;
yarn install;
yarn serve;
```

## After one of the above options
Navigate to localhost:8080
